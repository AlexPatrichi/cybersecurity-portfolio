# Cybersecurity Training Portfolio
I am a cybersecurity student building hands-on experience through labs, SOC simulations, and practical exercises.  

This repository documents my technical learning, combining:
- TryHackMe theory and practical labs
- Solent University Cyber Security Coursework
- Independent projects and exercises  

The focus is on developing real-world skills aligned with entry-level cybersecurity roles.

## Focus Areas
- SOC operations & security monitoring  
- Log analysis & incident response  
- SIEM tools & alert triage  
- Web & network security fundamentals  
- Tools & scripting for automation  

## Purpose
This portfolio demonstrates my progression from foundational knowledge to applied cybersecurity skills.

Each folder contains structured write-ups based on:

- TryHackMe labs
- University coursework (Solent University)
- Independent hands-on practice

It also reflects my ability to use Git and GitHub for documentation, version control, and technical organisation — key skills in real-world IT and security environments.

## Repository Structure
/cybersecurity-portfolio

   - TryHackMe theory and practical labs

1. Start with `/computer-science-basics/` for foundational lab exercises.
