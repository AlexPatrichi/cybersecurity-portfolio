## 🧠 Core Concepts Learned   
- Search engines index and organise vast amounts of information from the internet
- Basic searches often return broad or irrelevant results
- Advanced search techniques allow more precise and targeted queries

### Advanced Search Operators
1. **"exact phrase"**
- Searches for an exact word or phrase
- Example: "cyber security analyst"

2. **site:**
- Limits results to a specific domain name 
- Example: site:tryhackme.com success stories

3. **-**
- Excludes specific keywords from results 
- Example: pyramids -tourism

4. **OR**
- Finds webpages that contain either of the terms or phrases
- Example: cyber security OR information security

5. **filetype:**
- Searches for specific files (PDF, DOC, PPT, etc.) instead of web pages
- Example: filetype:pdf cyber security 

💡 These operators improve accuracy and reduce irrelevant results
💡 Most search engines treat multiple words as AND by default 

### Common Search Engines
- Google – Most widely used, powerful indexing and search operators  
- Bing – Alternative search engine with similar functionality  
- DuckDuckGo – Privacy-focused, does not track user activity  

💡 Different search engines may return different results, so using multiple sources improves accuracy
