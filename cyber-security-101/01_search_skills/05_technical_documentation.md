# Search Skills – TryHackMe and Solent University Cybersecurity Coursework 

Platform: TryHackMe    
Skill Level: Beginner / Foundation  
Focus Area: Technical Documentation

## 🎯 Objective 
- Understand the importance of using official technical documentation  
- Learn how to find and interpret reliable sources  
- Develop the ability to solve problems using documentation  

## 🧠 Core Concepts Learned   
- Official documentation is the most reliable source of technical information  
- It is created and maintained by the developers of a tool or technology  
- Includes setup guides, commands, configurations, and troubleshooting steps  

💡 A key skill in cybersecurity is knowing how to find and use official documentation  
💡 Documentation is usually more accurate and up-to-date than other online sources  

### Common Examples of Technical Documentation
#### Linux Manual Pages (man)
- Built-in documentation system on Linux and Unix-like systems  
- Provides detailed information about commands, system calls, and configuration files  

💡 Example:  
- Use `man ip` in a Linux terminal to view documentation for the `ip` command 
- Web versions of man pages can also be found by searching for "man ip" in a browser  

#### Microsoft Documentation
- Official documentation for Microsoft products and services  
- Includes setup guides, configuration steps, and troubleshooting  

💡 Example:  
- Used when working with Windows systems or Microsoft tools  

## 🛠️ Practical Skills Developed
- Finding and using official documentation to understand tools  
- Extracting relevant information from technical resources  
- Following setup guides and examples to solve problems  

## 🧰 Tools Used 
- TryHackMe platform 
- Solent University Cybersecurity Coursework 
- Mozilla Firefox (research and validation)

## 🔐 Security Relevance
- Helps ensure tools are used correctly and securely  
- Reduces the risk of misconfiguration and vulnerabilities  
- Supports accurate research during investigations   

## 📌 Lessons Learned 
⚠️ Not all online information is reliable — official documentation should be prioritised  
⚠️ Understanding documentation takes practice, especially with technical language  
⚠️ Good research skills are essential in cybersecurity
