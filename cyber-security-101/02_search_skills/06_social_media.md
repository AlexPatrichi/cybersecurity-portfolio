# Search Skills – TryHackMe and Solent University Cybersecurity Coursework 

Platform: TryHackMe    
Skill Level: Beginner / Foundation  
Focus Area: Social Media

## 🎯 Objective 
- Understand how social media is used in cybersecurity  
- Learn how information on social platforms can be analysed  
- Recognise risks associated with oversharing online  

## 🧠 Core Concepts Learned   
- Social media platforms contain large amounts of publicly available information  
- They can be used to gather intelligence about individuals and organisations   
- Oversharing personal information can create security risks  

💡 Social media is both a valuable source of information and a potential security weakness  

### Use in Cybersecurity
- Security professionals use social media to gather information about targets 
- Can help identify employees, technologies used, and company structure  

💡 Example: Identifying employees of a company through LinkedIn  

### Risks of Oversharing
- Personal information can be used for password recovery or social engineering  
- Attackers may collect details such as:
  - Birthdays  
  - Schools attended  
  - Pet names  

💡 Example: Answering security questions using publicly available information  

### Professional Use
- Social media can be used to stay updated with cybersecurity trends  
- Following security researchers, companies, and communities helps learning  

💡 Example: Following cybersecurity news and threat updates  

### Privacy Considerations
- Creating accounts with personal information can expose your identity  
- Temporary or secondary accounts can be used for exploration  

💡 Always be aware of what information you share online  

## 🛠️ Practical Skills Developed
- Identifying useful information from social media platforms  
- Recognising potential security risks from public data  
- Using social media as a research and learning tool  

## 🧰 Tools Used 
- TryHackMe platform 
- Solent University Cybersecurity Coursework 
- Mozilla Firefox (research and validation)

## 🔐 Security Relevance
- Helps identify potential information leaks  
- Supports investigations  
- Raises awareness of social engineering risks 

## 📌 Lessons Learned 
⚠️ Information shared online can be used against individuals or organisations  
⚠️ Social media is a powerful tool, but must be used responsibly  
⚠️ Maintaining privacy is essential in cybersecurity
