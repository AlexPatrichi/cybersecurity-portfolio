## 🧠 Core Concepts Learned

## Peripheral Devices
- Peripheral devices are external components connected to a computer system
- They allow users to interact with the system and extend its functionality

💡 Peripherals act as the bridge between the user and the computer system

## Types of Peripheral Devices

### 1. Input Devices (Data Entry)
- Used to send data or commands to the computer
- Convert user actions into digital signals 

#### Devices:
- **Keyboard**
    - Used for typing text and commands

- **Mouse** 
    - Pointing device used for navigation and interaction

- **Microphone** 
    - Captures audio input

- **Scanner**
    - Converts physical documents into digital formats 

- **Camera** 
    - Captures video input

### 2. Output Devices (Information Display)
- Used to present data from the computer in a human-readable format

#### Devices:
- **Monitor**
    - Displays visual information

- **Printer** 
    - Produce physical copies of digital documents

- **Speakers/ Headphones**
    - Produce audio output

- **Projector**
    - Displays computer output on a larger surface

### 3. Storage Peripherals
- External devices used for data storage, transfer, and backup
- Portable and commonly used to extend system storage

💡 External storage devices are important for backup and data portability

#### Devices:
- **USB flash drives**  
- **External hard drives**
- **Memory cards**

### 4. Network Peripherals
- Devices that enable communication between systems

#### Devices:
- **Network Interface Card (NIC)**
    - Allows a device to connect to a network (wired or wireless)

- **Router**   
    - Directs data packets between different networks

- **Modem**
    - Converts digital signals for transmission over the internet
    - Connects a local network to an Internet Service Provider (ISP)

- **Wi-fi** 
    - Enables wireless communication between devices over a network

- **Bluethoot**
    - Short-range wireless technology used for low-power devices

## Peripheral Connection
### Wired connections
- **Universal Serial Bus (USB)**
    - The most common interface for connecting peripherals 

- **HDMI and DisplayPort**
    - Used for connecting display devices
    - HDMI supports both audio and video signals 
    - DisplayPort is often used in high-performance setups

### Wireless connections
- **Bluethoot** 
    - Used for connecting low-power devices (keyboard, mouse, headset)
    - Offers short-range communication 

- **WIFI**
    - Enables wireless connection of peripherals like printers and external storage over a network

- **Near Field Communication (NFC)** 
    - Short-range communication technology for low-power communication between devices 
    - Used for contactless payments and quick data transfer

💡 Different connection types affect speed, reliability, and security 

## Device Drivers
- Software that allows the operating system to communicate with hardware devices
- Without drivers, peripherals may not function correctly

💡 Drivers act as translators between hardware and the operating system

## Plug and Play
- Ability of a system to automatically detect and configure new hardware
- Simplifies the process of connecting peripherals

## Troubleshooting Peripherals and Ports
**Common Issues & Solutions**
- **Driver Issues:** Missing or outdated drivers can prevent devices from functioning
    - Ensure all peripherals have up-to-date drivers installed

- **Physical Damage:** Ports or cables may be damaged or worn out
    - Inspect and replace faulty components

- **Connection Problems:** Interference, signal loss, or pairing problems can disrupt Bluetooth or Wi-Fi connections
    - Restart devices or reconnect

- **Power Issues:** Devices may not receive enough power
    - Try a different port or power source

## Peripheral Maintenance
- Dust and debris can affect performance and airflow
- Regular driver updates ensure compatibility and performance
- Disconnecting unused devices can improve system efficiency

## 🔐 Security Relevance
- Peripheral devices can introduce security risks (infected USB drives, interception risks)
- Input devices can be used for attacks such as keylogging
- Network peripherals can expose systems to external threats
- Untrusted devices may install malicious drivers or software

💡 External devices are common entry points for attackers and should not be trusted by default

## 📌 Lessons Learned
⚠️ Peripheral devices extend system functionality but also increase the attack surface  
⚠️ Not all external devices are safe. Connecting unknown hardware can introduce security risks  
⚠️ Understanding how devices interact with the system is essential for both usability and security
