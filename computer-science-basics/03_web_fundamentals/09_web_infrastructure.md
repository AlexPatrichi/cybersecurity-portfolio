# Web Fundamentals - TryHackMe and Solent University Cybersecurity Coursework 

Platform: TryHackMe   
Level: Beginner / Foundation  
Focus Area: Web Infrastructure and Components 

## 🎯 Objective 
- Understand the key components that support modern websites  
- Learn how infrastructure improves performance, scalability, and reliability  

## 🧠 Core Concepts Learned 
- Modern web applications rely on a combination of infrastructure components that work together to ensure performance, reliability, and security.

## Load Balancers 
- A system that distributes incoming traffic across multiple servers 
- This prevent any single server from becoming overloaded
- Perform **health checks** and stop sending traffic to failed servers  
   
### Purpose: 
- Improve performance by distributing traffic efficiently  
- Prevent server overload and potential crashes  
- Increase reliability and system availability  
- Support scalability to handle high volumes of users  
- Provide redundancy in case of server failure 
 
## Content Delivery Network (CDN)
- A global network of servers that deliver content from the closest location to the user  

### Benefits: 
- Faster load times  
- Reduced latency  
- Improved scalability  
- Increased reliability  

## Database
- A structured system used to store, manage, and retrieve data efficiently

### Purpose:
- Store and organise data for efficient retrieval and processing 
- Allow fast queries and updates  
- Support multiple users simultaneously  

### Examples:
- MySQL  
- PostgreSQL

## Web Application Firewall (WAF)
- Is a security system that monitors, filters, and blocks malicous HTTP/HTTPS traffic between users and web servers

### Purpose:
- Acts as a protective layer between the user and the server 
- Detect and block malicious requests  
- Protect against common web attacks 
    
## 🛠️ Practical Skills Developed
- Understanding how modern websites scale and handle traffic  
- Identifying infrastructure components in real-world systems   

## 🧰 Tools Used 
- Solent University Cybersecurity Coursework 
- TryHackMe platform  

## 🔐 Security Relevance
- Infrastructure components are critical for defence against attacks  
- Misconfigured systems can lead to performance issues or security vulnerabilities  

## 📌 Lessons Learned  
⚠️ Modern websites rely on multiple systems working together — not just a single server  
⚠️ Performance, scalability, and availability depend on proper infrastructure design  
⚠️ Misconfigurations in these components can introduce serious security vulnerabilities
